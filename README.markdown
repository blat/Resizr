Dependances
-----------------

* sinatra
* rmagick
* fileutils
* open-uri


Setup
-----------------

1. Install ruby and all dependancies (see above)

2. Launch application:

        ruby resizr.rb

3. Go to [http://localhost:4567](http://localhost:4567)
